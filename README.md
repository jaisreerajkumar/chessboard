# 🏰 Online Chessboard

An interactive chessboard built with HTML, CSS, and JavaScript. Play chess directly in your browser! This project focuses on a fully functional frontend chessboard, piece movement, and basic game rules.

### 🌟 Features

Fully responsive 8x8 chessboard

Standard chess pieces with drag-and-drop functionality

Basic legal moves for each piece

Highlighted possible moves when a piece is selected

Reset board functionality to start a new game

Visual feedback for captured pieces

Mobile-friendly design

### 🛠️ Technologies Used

HTML5 – Structure of the chessboard and pieces

CSS3 – Styling, responsive board, and animations

JavaScript (ES6) – Game logic, piece movement, and events

Optional Enhancements (not included by default):

Chess AI opponent

Move history and timer

Multiplayer over WebSocket

### 🚀 How to Run Locally

Clone the repository:

git clone https://github.com/your-username/online-chessboard.git


Open index.html in your preferred browser.

Start playing! Drag and drop pieces to make moves.

No installation or backend server is required — pure frontend project.

### 🎯 Usage

Click or drag a chess piece to see valid moves

Move pieces according to standard chess rules

Use the reset button to restart the game
